## Export Public Key

### Clone Repo

```
git clone https://github.com/shibaone/extract-pub-key.git
```

### Enter Directory

```
cd extract-pub-key
```

### Install Dependency

```
npm i
```

### Update index.js

```
Add your private key with 0x

const privateKeyString = '';//your privateKey with 0x
```

### Start Command

```
node index.js
```

### Result

You will get result in your terminal
